# Operational Checklist

## 1. Request Intake
- [ ] Create request file from template.
- [ ] Confirm requestor, team, and business purpose.
- [ ] Confirm proposed name follows `[AREA]-[TOPIC]-[YYYY]`.

## 2. Approval
- [ ] Validate ownership fields are complete.
- [ ] Validate at least two managers are assigned.
- [ ] Validate least-privilege access matrix.
- [ ] Record approver and approval date.

## 3. Drive Creation
- [ ] Create Shared Drive in Google Workspace.
- [ ] Apply baseline folders:
  - [ ] `00-admin/`
  - [ ] `10-operaciones/`
  - [ ] `20-proyectos/`
  - [ ] `30-reportes/`
  - [ ] `99-archivo/`
- [ ] Apply initial member roles.
- [ ] Record Shared Drive URL and creation timestamp.

## 4. Post-Creation Validation
- [ ] Verify no personal ownership for business-critical files.
- [ ] Verify external sharing controls are defined.
- [ ] Verify backup managers can administer access.

## 5. Quarterly Review
- [ ] Revalidate all memberships and roles.
- [ ] Remove expired temporary accesses.
- [ ] Record review date and outcomes in the request file.
