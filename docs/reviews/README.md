# Reviews Directory

This folder stores quarterly Shared Drive review records.

## Naming Convention
`YYYY-QN.md`

Examples:
- `2026-Q1.md`
- `2026-Q2.md`

## How to Create
Use:
`./scripts/new-quarterly-review.sh "<reviewer>" "[YYYY-QN]"`

If quarter is omitted, the script infers it from current date.
