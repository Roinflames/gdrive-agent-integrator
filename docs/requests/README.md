# Requests Directory

This folder contains generated and completed Shared Drive request records.

## Filename Convention
`YYYY-MM-DD-<slug>.md`

Examples:
- `2026-03-08-fin-presupuesto-2026.md`
- `2026-03-08-ops-incidentes-2026.md`

## Lifecycle
1. Create request with `scripts/new-drive-request.sh`.
2. Complete all required fields in the request file.
3. Record approval and creation details.
4. Keep quarterly review updates in the same file.
5. Do not delete approved requests; archive only when obsolete.
